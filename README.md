# HVAC Sales Forecasting & Business Intelligence Dashboard

## Overview
This project analyzes HVAC sales trends across regions and seasons to identify demand patterns and support better inventory planning. By integrating sales data with regional weather variables, the dashboard highlights how climate conditions influence HVAC product demand.

## Tools Used
- Power BI
- Excel

## Data Analysis
The analysis combined HVAC sales transaction data with regional weather variables to explore seasonal demand patterns. Interactive dashboards were developed to visualize sales performance across regions, product categories, and time periods.

Key analytical steps included:
- Integrating HVAC sales and regional climate data
- Identifying seasonal trends in product demand
- Comparing sales performance across geographic regions
- Evaluating correlations between temperature changes and HVAC sales

## Key Insights
- HVAC product demand increases significantly during temperature spikes
- Certain regions experience earlier seasonal demand increases
- Seasonal forecasting can help optimize inventory planning and reduce stock shortages
